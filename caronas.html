Sobre o Projeto

O Sistema de Caronas é uma aplicação de linha de comando (CLI) que simula uma plataforma como a BlaBlaCar. Ele permite que usuários se cadastrem, ofereçam caronas, e que outros usuários encontrem e reservem vagas nessas viagens. O projeto foi construído com foco em uma arquitetura de software modular e na manipulação de dados locais, sem a necessidade de um banco de dados externo.

Funcionalidades Principais

Gestão de Usuários

Cadastro, login e edição de perfil. O sistema valida os dados e garante que apenas maiores de 18 anos possam se registrar.

Recuperação de Senha

Um fluxo seguro para recuperação de senha, utilizando e-mail e data de nascimento para confirmação da identidade do usuário.

Oferta de Caronas

Motoristas podem cadastrar suas viagens, especificando origem, destino, data, horário, vagas disponíveis e valor por passageiro.

Busca e Reserva

Passageiros podem buscar caronas com base na origem e destino e realizar reservas de forma instantânea, debitando o número de vagas.

Relatórios para Motoristas

Funcionalidade para gerar um relatório financeiro detalhado das caronas oferecidas, com o total arrecadado.

Controle de Dados

O usuário tem total controle sobre seus dados, com a opção de editar seu perfil ou excluir sua conta permanentemente do sistema.
