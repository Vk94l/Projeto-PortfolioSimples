Política de Privacidade – Sistema de Caronas

Esta política de privacidade descreve como os dados pessoais dos usuários são coletados, usados e processados por este aplicativo, em conformidade com as melhores práticas de transparência e proteção de dados.

Tipos de Dados Coletados

Entre os tipos de dados pessoais que este aplicativo recolhe, por fornecimento direto do usuário, estão:

  • Dados de Conta: Nome completo, data de nascimento, e-mail e senha.
  • Dados de Carona: Origem, destino, data, horário, número de vagas e valor por passageiro.

Os dados pessoais são livremente fornecidos pelo usuário quando este se cadastra ou utiliza os serviços do aplicativo. O não fornecimento de determinados dados, como e-mail e senha, pode impossibilitar a prestação dos serviços de conta e reserva.

O usuário assume a responsabilidade pelos dados pessoais inseridos, garantindo que são precisos e verdadeiros.

Modo e Local de Processamento dos Dados

Método de Processamento

O controlador de dados processa os dados dos usuários de forma adequada e tomará as medidas de segurança apropriadas para impedir o acesso não autorizado, divulgação, alteração ou destruição não autorizada dos dados. O processamento é realizado utilizando o próprio software da aplicação (scripts em Python), seguindo procedimentos estritamente relacionados com os fins indicados.

Lugar

Os dados são processados e armazenados localmente na máquina onde o aplicativo está sendo executado, em um arquivo no formato JSON. Não há envio de dados para servidores externos.

Período de Conservação

Os dados são mantidos pelo tempo necessário para prestar o serviço solicitado pelo usuário (manter a conta ativa, histórico de viagens, etc.). O usuário pode solicitar ao controlador de dados (através da funcionalidade "Excluir Conta") que seus dados sejam removidos permanentemente.

O Uso dos Dados Coletados

Os dados relativos ao usuário são coletados para permitir que o proprietário do aplicativo forneça seus serviços, bem como para os seguintes propósitos:

  • Gestão de Contas: Para permitir o cadastro, login, edição de perfil e recuperação de senha.
  • Operação do Serviço de Caronas: Para que motoristas possam oferecer viagens e passageiros possam buscar, visualizar detalhes e reservar vagas.
  • Comunicação e Identificação: Para identificar o motorista e os passageiros de uma determinada viagem.
  • Relatórios Financeiros: Para permitir que os motoristas gerem relatórios sobre o rendimento de suas caronas.

Informações Adicionais sobre a Coleta e Processamento de Dados

Ação Jurídica

Os dados pessoais dos usuários podem ser utilizados para fins jurídicos pelo controlador de dados, em juízo ou nas etapas conducentes à possível ação jurídica decorrente de uso indevido deste serviço. O usuário declara estar ciente de que o controlador dos dados poderá ser obrigado a revelar os dados pessoais mediante solicitação das autoridades governamentais.

Logs do Sistema e Manutenção

Este aplicativo não coleta dados de uso ou logs de sistema automaticamente. Todas as informações armazenadas são fornecidas conscientemente pelo usuário durante a interação com o programa.

Os Direitos dos Usuários

Os usuários têm o direito de, a qualquer tempo, consultar seus dados (acessando seu perfil), verificar a sua exatidão ou pedir que sejam atualizados ou corrigidos (através da funcionalidade "Editar Perfil"), e solicitar o cancelamento e remoção de seus dados (através da funcionalidade "Excluir Conta").

Mudanças nesta Política de Privacidade

O controlador de dados se reserva o direito de fazer alterações nesta política de privacidade a qualquer momento. É recomendável que esta página seja consultada com frequência. Se um usuário não concordar com qualquer das alterações da política, ele deve cessar o uso deste serviço e pode solicitar a remoção dos seus dados pessoais.

Definições e Referências Jurídicas

  • Dados Pessoais (ou Dados): Quaisquer informações relativas a uma pessoa física que permitam sua identificação, como nome, e-mail, etc.
  • Usuário: A pessoa que utiliza este aplicativo, podendo atuar como motorista ou passageiro.
  • Controlador de Dados (ou Proprietário): A pessoa física ou jurídica que determina as finalidades e os meios de processamento de dados pessoais. Neste caso, o desenvolvedor e operador do software.
  • Este Aplicativo: A ferramenta de software (scripts Python) pela qual os dados pessoais do usuário são coletados.
  • Cookies: Não aplicável. Este aplicativo é um programa de linha de comando e não utiliza cookies.
